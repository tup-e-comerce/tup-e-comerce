# Proyecto TUP E-comerce

- **Nombre del proyecto:** tup-e-comerce
- **Tópico:** E-commerce y plataforma de gestión para salón de manicura.

El proyecto consiste en el desarrollo de una página web de comercio electrónico destinada a una casa especializada en servicios de manicura y venta de insumos. El objetivo principal es brindar una plataforma moderna, rápida y de fácil uso, que permita a los clientes:

- Ver los servicios de uñas ofrecidos.
- Comprar productos relacionados (esmaltes, acrílicos, herramientas, kits).
- Gestionar turnos.
- Realizar pagos online.
- Contactarse con la profesional.
